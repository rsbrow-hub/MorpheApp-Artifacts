## ✨ Features

- No root required
- Built-in ad-blocking & background play
- Return YouTube Dislike, SponsorBlock, and more
- Regular updates synced with latest Morphea patches
- All APKs are scanned on [VirusTotal](https://www.virustotal.com) before upload (link inside release)

---

## 📲 Download & Install

1. Go to the [**Releases**](../../releases) page.
2. Download the latest `.apk` file from the most recent release.
3. Enable **“Install from unknown sources”** in your device settings.
4. Install the APK and enjoy!

> ⚠️ If Google Play Protect shows a warning, press “Install anyway”. The APKs are safe, but they are not signed by the original developer.

## 🆚 MorpheaApp (NODOC-s) vs. Revanced – Comparison Table

| Aspect | **MorpheaApp (NODOC-s)** | **Revanced (Manual Method)** |
|--------|---------------------------|------------------------------|
| **Installation** | One‑click, pre‑patched APK. Just download and install. | Complex manual process: user must download the exact recommended APK version, then patch it using Revanced Manager. |
| **Technical skill required** | No technical knowledge needed. Beginner‑friendly. | Requires understanding of APK files, patching steps, and MicroG installation. |
| **Update frequency & process** | New pre‑patched APK released within hours of patch updates. Simply install the latest APK. | Every update requires re‑downloading the base APK and re‑patching. Sometimes the recommended version is outdated. |
| **Security scanning** | Every release APK is scanned on **VirusTotal** (70+ engines). Scan report is shared for transparency. | No centralised scan. Security entirely depends on the user’s own source of the base APK and the Manager app. |
| **Built‑in features** | Ad‑blocking, background play, SponsorBlock, Return YouTube Dislike, and more – all enabled out of the box. | Same patches are available, but the user must manually select and configure each one during patching. |
| **Overall user experience** | Fast, reliable, and hassle‑free. Ideal for users who don’t want to deal with complex steps. | Time‑consuming and sometimes frustrating. Some advanced users may find certain Revanced builds more stable, but the process is not for everyone. |
| **Trust & transparency** | All APKs are built from public Morphea patches; VirusTotal reports are linked in each release. | Source can be verified, but the final APK is built locally – no third‑party scan is provided by default. |
