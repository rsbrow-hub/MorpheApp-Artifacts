## 📲 How to Install (Step by Step Guide)

### ✅ Prerequisites
- Android 7.0 or higher
- Enable “Install from unknown sources” (Settings → Security)

### 📌 Steps
1. **Install MicroG** (skip if rooted)  
   Download and install the MicroG APK from the link above.
2. **Download MorpheApp APK**  
   Get the latest APK from the [Releases page](https://github.com/rsbrow-hub/MorpheApp-Artifacts/releases/latest). Both YouTube and Music variants are available.
3. **Install the APK**  
   Tap the downloaded file and install. If Google Play Protect warns you, click **“Install anyway”** (the file is safe but not signed by the original developer).
4. **Sign in**  
   Sign in with your Google account using MicroG. You’re now ready to enjoy completely ad‑free YouTube.

### ⚙️ Bonus: Import Our Settings
- Download the settings file from the link above and save it on your phone.
- Open MorpheApp and go to **Settings → Import/Export**.
- Select the file and import it.
- All premium features, the right theme, and optimized configurations will be applied instantly.
